# README

App hecha en Ruby on Rails 5.
Permite:

- *Listar usuarios*

- *Ver usuario*

- *Editar usuario*

- *Eliminar usuario*

- *Relacionar redes sociales a usuario*

- *Formulario de búsqueda autocompletable*
